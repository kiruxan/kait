## Codebase Patterns
- This is a Chrome Extension using Manifest V3
- Extension structure: extension/ directory with content/, background/, icons/, options/ subdirectories
- Content scripts in content/content.js and content/content.css
- Service worker in background/service-worker.js
- No TypeScript or build system - plain JavaScript

## 2026-01-31 - US-006
- What was implemented:
  - Created translateText() function in background/service-worker.js
  - Function accepts text, targetLanguage, and apiKey parameters
  - Makes POST request to https://api.x.ai/v1/chat/completions using grok-2-latest model
  - Uses prompt that auto-detects source language and returns only translated text
  - Parses response and returns translated text via {success, translation} object
  - Handles errors: 401 (invalid key), 429 (rate limit), network errors, timeout
  - Uses AbortController with 30 second timeout (REQUEST_TIMEOUT_MS = 30000)
  - Returns consistent response shape: {success: boolean, translation?: string, error?: string}
  - Exposes translateText via self.translateText for use by message handlers
- Files changed:
  - extension/background/service-worker.js (modified - added full translation service implementation)
- **Learnings for future iterations:**
  - Use AbortController with setTimeout for request timeouts in service workers
  - XAI API endpoint: https://api.x.ai/v1/chat/completions
  - XAI API uses standard OpenAI-style messages format with model, messages, temperature
  - Model to use: 'grok-2-latest' for translations
  - Temperature 0.3 works well for translation tasks (deterministic but not rigid)
  - Export functions in service worker using self.functionName for global scope
  - Always clear timeout in both success and catch paths to prevent memory leaks
